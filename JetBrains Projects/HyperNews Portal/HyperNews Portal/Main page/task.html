Description

Currently, the news articles can be accessed via direct links with identifiers that are known only to you. Let's collect the news articles on one page and display them all to our users.

Objectives

At the top of your main page, there should be a page header:

<h2>Hyper news</h2>

Below, you need to list your news articles and display the title, publication date, and a link to the article page for each.
Group the articles by date into unordered lists.

The most recent news should be at the top.

Add a link below to the page where users can add their own news:

target="_blank" target="_blank" href="/news/create/"

Example

Let's imagine that you have a JSON file with the following data:

[
  {
    "created": "2020-02-22 16:40:00",
    "text": "Text of the news 1",
    "title": "News 1",
    "link": 1
  },
  {
    "created": "2020-02-22 14:00:00",
    "text": "Text of the news 2",
    "title": "News 2",
    "link": 2
  },
  {
    "created": "2019-03-21 01:40:00",
    "text": "Text of the news 3",
    "title": "News 3",
    "link": 3
  },
  {
    "created": "2021-01-01 01:01:00",
    "text": "Text of the news 4",
    "title": "News 4",
    "link": 4
  }
]

If you start the application on your computer with the command python manage.py runserver, the main page will be available at the address localhost:8000/news/.

The expected result at localhost:8000/news/ is:

<h4>2021-01-01</h4>

<ul>
  <li><a target="_blank" href="/news/4/">News 4</a></li>
</ul>

<h4>2020-02-22</h4>

<ul>
  <li><a target="_blank" href="/news/1/">News 1</a></li>
  <li><a target="_blank" href="/news/2/">News 2</a></li>
</ul>

<h4>2019-03-21</h4>

<ul>
  <li><a target="_blank" href="/news/3/">News 3</a></li>
</ul>

Don't forget to add a link at the bottom of the page.

target="_blank" target="_blank" href="/news/create/"
