Objectives

Your task is to implement authentication on the website. You need to create two pages: one for new users to sign up and create their accounts and another to log in with their username and password.

Please use django.contrib.auth.forms.UserCreationForm for your signup handler and django.contrib.auth.forms.AuthenticationForm for the login handler.

If the login request is successful, redirect the user to the menu page.

If you start the application on your computer with the python manage.py runserver command, the signup page should be available at the address localhost:8000/signup and the login page should be at localhost:8000/login.

Please pay attention that the login and signup URLs do not have a slash at the end, so your URL patterns should not have it either. You can redirect GET requests to the right page with the help of RedirectView if you need to:

from django.views.generic import RedirectView
 
urlpatterns = [
   # some other patterns
   path('login/', RedirectView.as_view(url='/login'))
   path('signup/', RedirectView.as_view(url='/signup'))
]
