Description

The Hypercar service center provides three services to its customers:

  • Change oil
  • Inflate tires
  • Get diagnostic test

Your system needs to include a menu so customers can choose the service they require. The menu items should be the names of the different services. After clicking on an item, the customer should be redirected to a page where they can get a ticket for the correct service. To implement this, you will need to use special HTML elements to prepare the necessary links.

Objectives

Your next task is to make a menu page with three <a> elements, each with one of these href attributes: "/get_ticket/change_oil", "/get_ticket/inflate_tires", "/get_ticket/diagnostic". Here is an example of one of these elements:

<a target="_blank" href="/get_ticket/inflate_tires">Inflate tires</a>

As you can see, the link descriptions should match those shown in the bullet point list above.

The menu page should be available at <base_url>/menu, where <base_url> is the host. To combine a template with the HTTP handler, you can use the django.shortcuts.render function. To pass arguments to the template, you can add the dictionary to the render:

render(request, template_name, {'name_you_use_in_template': your_variable})
